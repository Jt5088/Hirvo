# Researcher 2 Findings: Modern, Minimalist & Clean Resume Templates

Focus: Modern, Minimalist, and Clean resume template designs from real-world sources. Emphasis on whitespace usage, typography precision, ATS compatibility, and contemporary aesthetics.

---

## Template 1: Jake's Resume (LaTeX/Overleaf)

**Source:** Overleaf / GitHub (github.com/jakegut/resume) -- ~5,000+ GitHub stars, one of the most-used tech resume templates in existence.

**Style Category:** Minimalist / Engineering-focused

**Layout Structure:**
- Single-column, full-width
- No sidebar, no graphics, no icons
- Section order: Education, Experience, Projects, Technical Skills
- Sections separated by horizontal rules (thin lines spanning full width)

**Typography:**
- Base: Computer Modern (LaTeX default), 11pt body
- Commented-out alternatives: FiraSans, Roboto, Noto Sans, Source Sans Pro (sans-serif); Cormorant Garamond, Charter (serif)
- Section headings: ALL CAPS, bold, ~13-14pt equivalent
- Job title/company: Bold for title, regular for company, right-aligned dates
- Bullet points: 11pt body text

**Spacing Analysis:**
- Document class: `letterpaper, 11pt`
- Margins: 0.5in left and right (reduced from default 1in via `\addtolength{\oddsidemargin}{-0.5in}`), ~0.5in top
- Text height extended by +1.0in beyond default
- Line spacing: Single (1.0) -- very tight
- Section gap: ~8-10pt between sections (rule + small vspace)
- Bullet spacing: ~2pt between items, tight packing
- Entry spacing: ~6pt between job entries

**Visual Elements:**
- Thin horizontal rules as section dividers (full-width lines)
- No color whatsoever -- pure black on white
- No icons, no graphics, no borders
- Dates right-aligned on same line as position title
- Bullet points use standard LaTeX itemize

**ATS Compatibility:**
- Perfect ATS score -- pure text, single column, standard headings, no tables/graphics
- LaTeX compiles to clean PDF text layer
- No multi-column parsing issues

**What Makes It Good:**
- Extreme information density without feeling cramped -- achieved through tight but consistent spacing
- The horizontal rules create clear section breaks without wasting vertical space
- Right-aligned dates create a clean visual rhythm
- Zero decoration means 100% content focus
- The template has become an industry standard in tech -- recruiters are accustomed to this format
- Proves that minimalism taken to its logical extreme can be highly effective

## Template 2: Resume.io "Berlin" Template

**Source:** Resume.io -- one of the largest resume builder platforms globally

**Style Category:** Modern Minimalist / Black & White

**Layout Structure:**
- Single-column primary layout
- Clean header block with name and title
- Sections flow top-to-bottom: Summary, Experience, Education, Skills
- No sidebar -- all content uses full page width

**Typography:**
- Primary font: Sans-serif (Lato or similar clean grotesque)
- Name: ~22-24pt, bold, black
- Section headings: ~14-16pt, bold, uppercase or title case with subtle letter-spacing
- Body text: 10-11pt, regular weight
- Company/title line: Bold for role, regular for company, separated by a pipe or comma

**Spacing Analysis:**
- Margins: ~0.75in all sides (balanced, not too tight)
- Line spacing: 1.15 within body text
- Section gap: ~18-24pt between major sections
- Entry gap: ~10-12pt between individual entries
- Bullet point spacing: ~4-6pt between bullets
- Header to first section: ~20-24pt

**Visual Elements:**
- Striking black-and-white palette -- no color accents at all
- Section headings may use a thin underline or subtle horizontal rule
- Clean geometric feel from consistent alignment
- No icons, no progress bars, no graphics
- Generous whitespace around header area

**ATS Compatibility:**
- Excellent -- single column, standard section headings, no complex formatting
- PDF export maintains clean text layer
- No tables, text boxes, or multi-column layouts to confuse parsers

**What Makes It Good:**
- The all-black-and-white approach creates maximum contrast and a bold, confident impression
- Feels premium through restraint -- the absence of decoration IS the design
- The ~0.75in margins give comfortable breathing room without wasting space
- Works equally well on screen and in print
- The template proves that color is unnecessary for a memorable, professional resume

## Template 3: Enhancv Single Column Template

**Source:** Enhancv -- premium resume builder platform

**Style Category:** Clean / Professional / Modern

**Layout Structure:**
- Single-column, full-width content
- Header: Name, title, contact info in a clean horizontal arrangement
- Sections: Summary, Experience, Education, Skills, Certifications
- Each section takes full page width with clear separation

**Typography:**
- Primary font: Rubik or similar modern geometric sans-serif
- Name: ~20-24pt, weight 600-700
- Section headings: ~14pt, bold, with accent color optional
- Body text: 10-11pt, regular weight, ~1.15-1.2 line height
- Date/location: 10pt, lighter weight or gray color

**Spacing Analysis:**
- Margins: Adjustable in editor, default ~0.7-0.8in all sides
- Line spacing: 1.15-1.2 for body text
- Section gap: ~20-28pt between sections
- Entry gap: ~12pt between entries within a section
- Bullet spacing: ~5-6pt between bullet points
- Section heading to first entry: ~8-10pt
- Overall whitespace ratio: ~30-35% of page

**Visual Elements:**
- Optional accent color (single color, applied to section headings or a thin left border)
- Clean horizontal lines between sections (not full-width, centered with fade)
- No icons by default -- clean text-only approach
- Optional subtle background pattern (dots, lines) that can be turned off
- Contact info uses simple text separators (pipes or bullets)

**ATS Compatibility:**
- Strong ATS performance -- single column eliminates parsing confusion
- Standard heading names
- No tables or complex layout structures
- Text exports cleanly to .txt format

**What Makes It Good:**
- The customization system lets you dial minimalism up or down while maintaining the same clean grid
- Default state is very clean -- you have to actively add decoration
- The adjustable margin system means you can optimize for content density vs. whitespace
- Section heading treatment (bold + optional color accent) creates hierarchy without visual noise
- Feels more contemporary than Jake's Resume while maintaining similar ATS reliability

## Template 4: Canva Swiss-Inspired Minimalist Template

**Source:** Canva / Etsy (Swiss-inspired resume bundles)

**Style Category:** Swiss International Typographic Style / Grid-Based Minimalist

**Layout Structure:**
- Grid-based layout inspired by the International Typographic Style
- Three layout variants in typical bundles: minimalist grids, asymmetrical balance, layered designs
- Primary variant: Two-column with narrow left column (30%) for dates/labels and wide right column (70%) for content
- Header area uses full width with name and contact on a clean baseline

**Typography:**
- Primary font: Helvetica Neue or Inter (staying true to Swiss origins)
- Name: 24-28pt, weight 300-400 (light weight -- distinctly Swiss)
- Section headings: 11-12pt, ALL CAPS, weight 700, letter-spacing 0.1-0.15em
- Body text: 10pt, regular weight, line-height 1.4-1.5
- Dates: 9-10pt, regular or light weight, same font family
- Contrast created through weight and case, not size variation

**Spacing Analysis:**
- Margins: 0.75in top/bottom, 0.6-0.75in left/right
- Grid columns: 30/70 split with ~16-20pt gutter
- Line spacing: 1.4-1.5 (more generous than most -- Swiss style values readability)
- Section gap: ~24-32pt between sections
- Entry gap: ~14-16pt between entries
- Bullet spacing: ~6pt between bullets
- Heading to content: ~6-8pt
- Overall whitespace: ~35-40% (highest of templates analyzed)

**Visual Elements:**
- Minimal color -- typically black text on white, with ONE accent color used sparingly (a thin rule, a section heading color)
- Thin horizontal rules (0.5pt) as section dividers
- Flush-left, ragged-right text alignment (key Swiss principle)
- No icons, no decorative elements
- The grid alignment itself IS the visual design -- everything snaps to invisible grid lines
- Generous margins create a "frame" effect around content

**ATS Compatibility:**
- Good with single-column variant; the two-column variant can cause issues with some older ATS
- Clean text layer, standard fonts
- No graphics or complex elements

**What Makes It Good:**
- The light-weight name heading (300-400 weight) is distinctly modern and elegant -- most templates go bold, this goes light
- The 30/70 column split creates a natural reading rhythm: scan dates on left, read content on right
- Higher line-height (1.4-1.5) makes body text exceptionally readable
- ALL CAPS section headings with generous letter-spacing create section breaks through typography alone -- no heavy dividers needed
- The grid-based approach ensures mathematical harmony in all spacing relationships
- Feels timeless rather than trendy -- Swiss design has been "modern" since the 1950s

## Template 5: Rezi Standard ATS Template

**Source:** Rezi.ai -- AI-powered resume builder focused on ATS optimization

**Style Category:** Clean / ATS-Optimized / Functional Minimalist

**Layout Structure:**
- Strict single-column layout (no sidebar, no multi-column sections)
- Header: Name centered or left-aligned, contact info on one line below
- Linear section flow: Summary, Experience, Education, Skills
- Skills section uses comma-separated inline lists (not multi-column grids)

**Typography:**
- Primary font: Arial, Calibri, or similar system sans-serif
- Name: 18-22pt, bold
- Section headings: 12-14pt, bold, sometimes with thin underline
- Body text: 10.5-11pt
- Bullet points: Same size as body, standard round bullets
- Dates: Right-aligned, same size as body, regular weight

**Spacing Analysis:**
- Margins: 0.5-0.75in (tighter than average to maximize content space)
- Line spacing: 1.0-1.15 (tight, optimized for density)
- Section gap: ~14-18pt between sections
- Entry gap: ~8-10pt between entries
- Bullet spacing: ~3-4pt between bullets
- Section heading underline gap: ~4pt below heading
- Overall whitespace: ~25-28% (on the lower end -- prioritizes content volume)

**Visual Elements:**
- Extremely minimal -- essentially a well-formatted Word document
- Thin underlines below section headings (the only decorative element)
- No color (black text on white)
- No icons, charts, progress bars, or graphics
- Standard round or dash bullet points
- Clean baseline alignment throughout

**ATS Compatibility:**
- Maximum ATS compatibility -- this is the template's primary design goal
- All Rezi templates are "verified ATS-friendly"
- Standard fonts, proper tagging, optimized margins
- No formatting that could confuse any ATS parser
- Clean text extraction guaranteed

**What Makes It Good:**
- This is the control group -- the "zero decoration" approach to resume design
- Proves that ATS optimization and minimalism are naturally aligned
- Tighter margins (0.5-0.75in) allow more content without feeling cramped, because the single-column layout provides enough horizontal breathing room
- The thin heading underlines are the perfect amount of visual structure -- they create section breaks without any decorative overhead
- Right-aligned dates create a two-point reading anchor (left: content, right: timeline)
- Works universally -- no industry, role, or ATS system where this template would be rejected

## Template 6: Figma "Clean Minimal Resume / CV Template"

**Source:** Figma Community (figma.com/community/file/1192162451838440484)

**Style Category:** Designer-Minimal / Portfolio-Adjacent

**Layout Structure:**
- Single page, primarily single-column
- Large header area with name, title, and brief bio
- Two-column section for skills/contact (narrow left) and experience (wide right)
- Bottom section returns to single-column for education/certifications

**Typography:**
- Primary: Inter or similar neo-grotesque sans-serif
- Name: 28-36pt, weight 700-800 (large and bold -- makes a statement)
- Section labels: 10-11pt, ALL CAPS, letter-spacing 0.1em, weight 600
- Body text: 10-11pt, weight 400, line-height 1.5
- Subheadings (company/role): 12pt, weight 600

**Spacing Analysis:**
- Margins: ~40-48px (roughly 0.55-0.67in at 72dpi)
- Header area: ~80-100px tall with generous padding
- Section gap: ~32-40px between major sections
- Entry gap: ~16-20px between entries
- Bullet spacing: ~8px between bullets
- Column gutter: ~24-32px in two-column areas
- Overall whitespace: ~35-40%

**Visual Elements:**
- Monochromatic -- black/dark gray text on white
- Thin 1px horizontal lines as section dividers
- No icons or decorative graphics
- Name functions as a design element through sheer size
- Clean geometric alignment gives the page visual structure
- Contact info often uses minimal inline separators

**ATS Compatibility:**
- Moderate -- the two-column skills section may cause issues with some ATS
- When exported as PDF, text layer is generally clean
- The large header may cause name/title parsing variations

**What Makes It Good:**
- The oversized name treatment (28-36pt) turns typography into the primary design element
- Higher line-height (1.5) in body text creates exceptional readability
- The mix of single and two-column areas adds visual interest while keeping the main content (experience) in a clean single column
- Designed in Figma, so spacing relationships are pixel-perfect and mathematically intentional
- Demonstrates that "minimal" doesn't mean "boring" -- scale contrast alone creates visual hierarchy

### Font Recommendations for Modern/Minimalist
1. **Inter** -- The modern default. Clean, readable, excellent at small sizes
2. **Helvetica Neue** -- Swiss heritage, timeless authority
3. **Calibri** -- Microsoft default, universally available, surprisingly well-designed
4. **Lato** -- Warm humanist sans-serif, approachable yet professional
5. **Rubik** -- Geometric with rounded terminals, feels contemporary

### Key Design Patterns Across All Templates
1. **Single-column dominance** -- 5 of 6 templates are primarily single-column
2. **Black on white** -- No template requires color to function
3. **Horizontal rules** -- The most common (and effective) section separator
4. **Right-aligned dates** -- Creates a clean timeline column without a formal sidebar
5. **Type scale > decoration** -- All templates create hierarchy through font size/weight, not ornamentation
6. **Tight body, generous sections** -- Body text is tight (1.0-1.15), but section gaps are generous (18-24pt)

## Template 7: Standard Resume "Pender" (Typography-Forward Minimalist)

**Source:** StandardResume.co -- modern resume builder trusted by 100,000+ job seekers, offering 12 professionally designed templates

**Style Category:** Typography-Forward Minimalist / Whitespace-Driven

**Layout Structure:**
- Single-column, full-width content
- Oversized summary section at the top -- designed to capture attention before the recruiter scrolls
- No borders, no separators, no icons whatsoever
- Sections flow linearly: Contact, Summary (prominent), Experience, Education, Skills
- Optional photo placement (headshot or logo)
- Supports hybrid, functional, and chronological ordering

**Typography:**
- Primary font: Inter by Rasmus Andersson (all weights 100-900 available)
- Inter's tall x-height aids mixed-case and lowercase readability
- Contextual OpenType features enabled for refined typesetting
- Name: ~24-28pt, weight 700-800
- Section headings: ~12-14pt, weight 600, title case
- Body text: 10.5-11pt, weight 400, line-height ~1.4-1.5
- Job titles/companies: ~12pt, weight 600
- Dates/metadata: 10pt, weight 400, lighter color (gray)

**Spacing Analysis:**
- Margins: Responsive -- 16px on mobile scaling to 72px on desktop (roughly 0.22in to 1.0in at 72dpi)
- Section gaps: 24-40px between major sections
- Entry gaps: 16-24px between individual entries
- Bullet spacing: ~6-8px between bullets
- Summary section: Significantly larger font size than body, with generous padding above and below (~32-48px)
- Overall whitespace: ~38-42% -- among the highest of any template analyzed
- The template uses whitespace as its primary design tool -- there is literally nothing else

**Visual Elements:**
- Zero decoration. No lines, no rules, no borders, no icons, no color accents
- "Serene reading experience through exceptional whitespace use" (StandardResume's own description)
- The absence of all visual elements IS the design statement
- Larger job description font size "encourages reading beyond scanning" -- a deliberate UX choice
- Clean baseline alignment throughout
- Typography and spacing do 100% of the visual hierarchy work

**ATS Compatibility:**
- Excellent -- single column, standard fonts, standard section headings
- No tables, no multi-column, no graphics to confuse parsers
- Inter is a widely-supported web font with clean text extraction
- Clean PDF export with proper text layer

**What Makes It Good:**
- This is the most radical "design through absence" template in the research set. Where Jake's Resume is minimal because it prioritizes density, Pender is minimal because it prioritizes READING. The oversized summary section is a deliberate UX bet: force the recruiter to read your pitch before they scan your bullet points.
- Inter at multiple weights (400/600/700/800) creates a full type hierarchy without ever needing a second font
- The responsive margin system (16-72px) is sophisticated -- most templates use fixed margins
- The tall x-height of Inter makes 10.5pt body text feel like 11-12pt in other fonts, allowing more content per page without sacrificing readability
- Proves that a resume can be genuinely beautiful with zero visual elements -- just font, weight, and space

## Template 8: Standard Resume "Cordova" (Serif/Sans-Serif Editorial)

**Source:** StandardResume.co

**Style Category:** Editorial / Serif-Sans Pairing / Designer-Friendly

**Layout Structure:**
- Single-column, full-width content
- Clean header with name and contact details
- Sections flow linearly: Summary, Experience, Education, Skills
- Editable color scheme (accent color applied to headings or name)
- Minimalist single-column -- no sidebar, no multi-column areas

**Typography:**
- Heading font: EB Garamond by Georg Duffner (serif -- elegant, high stroke contrast)
- Body font: Inter by Rasmus Andersson (sans-serif -- clean, modern)
- This is the ONLY template in our research set that uses a serif/sans-serif pairing
- Name: ~26-32pt, EB Garamond, weight 500-600 (the serif makes a strong first impression)
- Section headings: ~13-15pt, EB Garamond, weight 600, potentially with accent color
- Body text: 10.5-11pt, Inter, weight 400, line-height ~1.4
- Job titles: ~12pt, Inter, weight 600
- Dates: 10pt, Inter, weight 400

**Spacing Analysis:**
- Margins: Responsive 16-72px (same system as Pender)
- Section gaps: ~24-32px between sections
- Entry gaps: ~16-20px between entries
- Bullet spacing: ~6-8px
- Heading to first entry: ~8-10px
- Overall whitespace: ~32-36%

**Visual Elements:**
- The serif/sans-serif contrast is itself the primary visual element
- Optional accent color (applied to name or section headings) -- single color, used sparingly
- No icons, no borders, no progress bars, no decorative graphics
- EB Garamond's high stroke contrast and delicate curves create visual interest through the letterforms themselves
- The contrast between Garamond headings and Inter body text creates a natural "editorial" feel -- like a well-typeset magazine

**ATS Compatibility:**
- Strong -- single column, standard section names, clean text layer
- EB Garamond and Inter are both Google Fonts with excellent PDF embedding
- Some older ATS may render Garamond differently, but text extraction remains clean
- No layout elements that could confuse parsers

**What Makes It Good:**
- The serif/sans-serif pairing is the most typographically sophisticated approach in our entire research set. It signals design awareness without any actual "design" elements -- pure typographic craft.
- EB Garamond headings feel authoritative and editorial (like a New York Times byline), while Inter body text feels modern and efficient. The tension between these two creates visual energy.
- This template is the answer to "I want my resume to look designed, but I work in a conservative industry." The font pairing alone does the work that other templates achieve with icons, sidebars, and color blocks.
- Recommended font pairings like EB Garamond + Inter, Merriweather + Lato, and Georgia + Helvetica are well-established in web design -- this template brings that web typography practice to resume design.
- For Hirvo, this represents the "editorial/creative professional" archetype -- the template a content strategist, product marketer, or UX writer would choose.

## Template 9: Swiss Minimal CV (LaTeX/GitHub Open Source)

**Source:** GitHub (github.com/obk/swiss-minimal-cv) -- open-source LaTeX template explicitly inspired by Swiss International Typographic Style

**Style Category:** Swiss International Typographic Style / Grid-Based Minimalist / Open Source

**Layout Structure:**
- Single-column layout, strict grid alignment
- Header section with name and contact
- Clean section divisions using typographic hierarchy (not graphic elements)
- Custom LaTeX commands for consistent section formatting (`\jobtitle{}`, etc.)
- ATS-optimized single-column structure

**Typography:**
- Primary font: Helvetica / Arial (the canonical Swiss design typefaces)
- Three-color text hierarchy:
  - `mainblack`: #000000 -- primary text, headings
  - `darkgray`: #333333 -- subtitles, job metadata, dates
  - `lightgray`: #555555 -- accent elements, tertiary info
- Section headings: Bold, uppercase (via `\MakeUppercase{}`), creating hierarchy through case and weight
- Body text: Regular weight Helvetica/Arial
- Styling uses `\textbf{}` for bold emphasis, `\textit{}` for dates/locations

**Spacing Analysis:**
- Margins: left=1.5cm, right=1.5cm, top=1.25cm, bottom=1.25cm (approximately 0.59in left/right, 0.49in top/bottom)
- Header spacing: 0.2cm (2mm / ~5.7pt) vertical gap within header elements
- These are tighter-than-average margins, prioritizing content density while maintaining Swiss design readability
- The geometry package allows easy margin customization
- Section gaps: Controlled via LaTeX spacing commands -- estimated ~12-16pt between sections based on Swiss design conventions
- Line spacing: Default LaTeX single spacing (~1.0-1.1)

**Visual Elements:**
- Three-tone gray scale (#000, #333, #555) creates depth without color
- NO decorative elements -- no rules, no lines, no icons, no borders
- The grid alignment and consistent use of custom commands ensure mathematical harmony
- Hierarchy created entirely through: font weight (regular vs bold), text case (mixed vs uppercase), and gray value (black vs dark gray vs light gray)
- Black/dark-gray color scheme optimized for clean printing
- Centralized color definitions at document beginning for easy theming

**ATS Compatibility:**
- Excellent -- single column, standard Helvetica/Arial fonts, no graphics
- LaTeX compiles to clean PDF with proper text layer
- Standard section names enforced by custom commands
- No tables, no multi-column parsing risks

**What Makes It Good:**
- This is the most rigorous implementation of Swiss design principles in a resume template. Where the Canva Swiss-inspired template (Template 4) applies Swiss aesthetics broadly, this template encodes them in LaTeX with mathematical precision.
- The three-tone gray hierarchy (#000/#333/#555) is a sophisticated alternative to the binary black-on-white approach. It creates depth and reading priority without any color at all -- pure luminance contrast.
- Tight margins (1.5cm / 0.59in) combined with Helvetica's inherent readability allow high content density without feeling cramped.
- As an open-source LaTeX template, the exact spacing values are inspectable and reproducible -- there's no ambiguity about the design intent.
- The custom command approach (`\jobtitle{}`, etc.) enforces consistency -- every entry looks the same because the formatting is abstracted into commands, not applied manually.
- For Hirvo, this validates the Swiss approach with a concrete, code-verified implementation. The three-tone gray system is particularly worth adopting.
